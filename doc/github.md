### 导航和页面切分组件库
- [Scene](https://github.com/bytedance/scene/blob/master/README_cn.md)

  是一个基于View的轻量级导航和页面切分组件库.
  1. 简单方便的页面导航和栈管理，支持MultiStack
  2. 完善的生命周期的管理和分发
  3. 可以更简单的实现复杂的过场动画
  4. 支持对Activity和Window属性的修改和恢复
  5. 支持页面之间拿返回值，支持在Scene中申请权限
  6. 支持页面销毁时保存状态和恢复
